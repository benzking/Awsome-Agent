## 📚 分析报告

### knowledge-pipline

**Knowledge Pipeline** — 将 PDF、图片、视频、Word、Excel 等多格式文档编译成结构化 Markdown 维基 + 交互式知识图谱的 Claude Code 技能。核心特点是增量知识积累、跨文档矛盾检测和深度推理链。

👉 [查看完整分析报告](./llm-wiki/knowledge-pipline/report.md)

---

### omni-article-markdown

**Omni-Article-Markdown（墨探）** — 将任意网页文章转换为 Markdown 格式的 CLI 工具，支持 34 个专用 Extractor 覆盖 Medium、CSDN、掘金、公众号、知乎等主流平台，三模块 Reader→Extractor→Parser 架构，支持 Docker 和 Playwright 反爬虫。

👉 [查看完整分析报告](./llm-wiki/omni-article-markdown/report.md)
